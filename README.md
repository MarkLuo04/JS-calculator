# JS Calculator

**JS Calculator** is a minimalistic, yet fully functional calculator built on JavaScript that supports:

- **Basic Arithmetic**: Addition, subtraction, multiplication, division  
- **Square Root**  
- **Decimal and Exponential Notation**
- **Operation chaining**
- **Keyboard Support**
